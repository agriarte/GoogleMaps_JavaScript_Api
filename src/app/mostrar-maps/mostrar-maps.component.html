Comienza componente que muestra mapa
Desde https://console.cloud.google.com en Biblioteca de Api se tiene que habilitar Google Maps JavaScript Api
y generar una key que nos autorice a utilizar googlemaps
En el proyecto Ionic, buscar archivo index.html en src/ y añadir esto justo dentro del HEAD:
<script src='https://maps.googleapis.com/maps/api/js?libraries=places&key=MI_KEY_AQUI'> </script>
